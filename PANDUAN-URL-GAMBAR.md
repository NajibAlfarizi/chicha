## âœ… Solusi: Cara Mendapatkan URL Gambar yang Benar

### **Metode 1: Copy Image Address (Tercepat)**

1. **Buka halaman produk** (Shopee, Tokopedia, dll)
2. **Klik kanan pada gambar produk**
3. **Pilih "Copy Image Address"** atau **"Salin Alamat Gambar"**
4. **Paste URL** ke form produk

**Contoh URL yang benar:**
```
https://cf.shopee.co.id/file/id-11134207-7r98o-lxyz123.jpg
https://images.tokopedia.net/img/cache/500-square/product-1/2023/12/15/xyz.jpg
https://i.imgur.com/abc123.jpg
```

---

### **Metode 2: Upload ke Image Hosting (Recommended)**

#### **A. Menggunakan Imgur (Gratis)**

1. Pergi ke https://imgur.com
2. Klik **"New post"** atau **"Upload"**
3. Upload gambar produk Anda
4. Setelah upload, klik kanan gambar â†’ **"Copy Image Address"**
5. Paste URL ke form produk

**Contoh URL Imgur:**
```
https://i.imgur.com/abc123.jpg
```

#### **B. Menggunakan ImgBB (Gratis)**

1. Pergi ke https://imgbb.com
2. Klik **"Start uploading"**
3. Upload gambar produk
4. Copy **"Direct link"** (bukan "HTML full linked")
5. Paste URL ke form produk

**Contoh URL ImgBB:**
```
https://i.ibb.co/abc123/image.jpg
```

#### **C. Menggunakan Cloudinary (Gratis + Fitur Lebih)**

1. Daftar di https://cloudinary.com (gratis)
2. Upload gambar ke Media Library
3. Copy URL gambar
4. Paste ke form produk

**Contoh URL Cloudinary:**
```
https://res.cloudinary.com/your-account/image/upload/v123/product.jpg
```

---

### **Metode 3: Gunakan Gambar dari Website Resmi**

Jika produk punya website resmi (Samsung, Apple, dll):
1. Buka website resmi produk
2. Cari gambar produk berkualitas tinggi
3. Klik kanan â†’ **"Copy Image Address"**
4. Paste URL ke form

## ğŸ¯ Rekomendasi untuk Chicha Mobile

Untuk hasil terbaik:

1. **Upload gambar sendiri** ke Imgur/ImgBB
   - Kontrol penuh atas gambar
   - Tidak bergantung pada website lain
   - Gambar tidak hilang jika sumber dihapus

2. **Gunakan gambar berkualitas tinggi**
   - Minimal 800x800 px
   - Format JPG atau PNG
   - Ukuran file < 500KB (agar cepat load)

3. **Konsisten dengan style**
   - Background putih/transparan
   - Produk di tengah
   - Pencahayaan baik
